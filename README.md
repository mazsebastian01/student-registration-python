# Student Registration Program

> Description

This is a simple program made with Python.
The program lets the user register students and save their names in a list.

It asks the user to type the name of the student and then it saves the name.
The user can add more students if they want.

**Features**

	•	Register students
	•	Save student names in a list
	•	Add more than one student

**Requirements**

You need Python 3 installed on your computer.

How the program works

	1.	The program asks for the student’s name.
	2.	The name is saved in a list called students.
	3.	The program asks if the user wants to add another student.
	4.	If the user says yes, the program continues.
	5.	If the user says no, the program stops.
